Widget system ING app

Our project for ING focused on creating helpful tools in their app to make it easier for customers to manage their money better. We studied and designed features that would guide each user, no matter their financial situation, toward being more financially secure.

Service

Research, prototyping, UX Design, UI Design

Client

ING

Tools

Figma

How might we facilitate behavioural change to achieve financial health amongst ING customers using various widgets?

Achievable financial goals

Widgets in the ING app allows users to set up a budget with trusted recommendations based on insights from ING users. Personalized goals can be set to help achieve financial milestones with ease.

More insight in your finances

ING users now receive periodic insights on their screen, such as monthly overviews to track expenses and savings. Fixed charges can be easily managed, providing users with clear and accessible financial information.

Saving tips

ING users now receive saving tips directly on their screens, such as reminders to set aside a portion of their income, alerts for unpaid requests, and suggestions for managing expenses like reviewing subscription usage. These proactive insights help users stay on top of their finances.
