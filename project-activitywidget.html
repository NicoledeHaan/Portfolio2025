Activity dashboard

To improve transparency and provide users with better insights into their account activitie I designed an activity widgets and dashboard for MediaLab users to track file upload statuses and gain insights into their account activities. It provides details such as user activity, storage usage, and file uploads, ensuring users stay informed and in control.

Service

Prototyping, UX Design, UI Design

Client

MediaLab

Tools

Adobe XD

How can MediaLab enable users to easily track file upload statuses and gain insights into their account activities, while presenting this information in a clear way?

Dashboard

An activity dashboard, provide project owners clear insights into their current activities from them and their team users about file uploads, storage usage, and account activity etc.

Widgets

Compact widgets for MediaLab users that provide real-time visibility into uploads, storage and activity, now viewable from anywhere, including outside the dashboard.

Notifications

Small notifications, always visible at the bottom left of the screen, provide real-time updates on active uploads, push queue and video queue. Users can hide these notifications by preference.
